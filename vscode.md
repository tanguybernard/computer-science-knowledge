


## Java

Extension Pack for Java


## Shortcut

Cmd + b : masquer la barre de gauche

Cmd + j : masquer la barre inférieur


## Recherche Global a activer par defaut

1. Ouvrez la Palette de Commandes avec ce raccourci :

Cmd ⌘ + Shift ⇧ + P

2. Dans la barre qui apparaît, tapez (ou copiez) ceci : Keyboard Shortcuts JSON

3. Ajouter :

    {
      "key": "cmd+shift+f",
      "command": "search.action.openNewEditor"
    }
